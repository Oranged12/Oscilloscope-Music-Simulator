# Oscilloscope-Music-Simulator
An Oscilloscope Simulator on Python to display Oscilloscope music!

This code works on Python Version 3.8 and up.

## Libraies include
1. Pygame
2. Numpy
3. Soundfile
4. imageio
5. imageio-ffmpeg

You **must** Have (FFMPEG)[https://www.ffmpeg.org/] installed for this code to work.
